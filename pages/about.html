HOW TO PLAY

Predict the colors

This is a game where a code (sequence of colors) is automatically randomly selected and hidden. Then the code braker get to to have some chances and guess the code colour. You can choose your level Beginner, Intermediate and Advanced. If you fail do not worry and Lets have some fun!!
